Exercises

1. Create a variable `name` and assign your name to it. Print the value of `name`.

name = "Your Name"\nprint(name)

This code creates a variable name and assigns it the value "Your Name". The print() function then outputs the value of the name variable.

2. Create two variables `a` and `b`, assign them the values 5 and 10 respectively. Print their sum.

a = 5\nb = 10\nprint(a + b)

This code creates two variables a and b with values 5 and 10 respectively. The print() function then outputs their sum.

3. Create a variable `pi` and assign it the value 3.14. Print the value of `pi`.

pi = 3.14\nprint(pi)

This code creates a variable pi and assigns it the value 3.14. The print() function then outputs the value of the pi variable.

4. Create a variable `is_student` and assign it the value `True`. Print the value of `is_student`.

is_student = True\nprint(is_student)

This code creates a variable is_student and assigns it the value True. The print() function then outputs the value of the is_student variable.

5. Create a list `fruits` containing the values "apple", "banana", and "cherry". Print the list.

fruits = ["apple", "banana", "cherry"]\nprint(fruits)

This code creates a list fruits containing the values "apple", "banana", and "cherry". The print() function then outputs the list.

6. Create a dictionary `person` with keys "name", "age", and "city" and assign them appropriate values. Print the dictionary.

person = {"name": "John", "age": 30, "city": "New York"}\nprint(person)

This code creates a dictionary person with keys "name", "age", and "city" and assigns them appropriate values. The print() function then outputs the dictionary.

7. Create a variable `x` and assign it the value 10. Create another variable `y` and assign it the value of `x` multiplied by 2. Print the value of `y`.

x = 10\ny = x * 2\nprint(y)

This code creates a variable x and assigns it the value 10. Another variable y is created and assigned the value of x multiplied by 2. The print() function then outputs the value of y.

8. Create a variable `text` and assign it the value "Hello, World!". Print the length of `text`.

text = "Hello, World!"\nprint(len(text))

This code creates a variable text and assigns it the value "Hello, World!". The print() function then outputs the length of the text variable using the len() function.

9. Create a variable `numbers` and assign it a list of numbers from 1 to 5. Print the first element of the list.

numbers = [1, 2, 3, 4, 5]\nprint(numbers[0])

This code creates a variable numbers and assigns it a list of numbers from 1 to 5. The print() function then outputs the first element of the list using numbers[0].

10. Create a variable `temperature` and assign it the value 98.6. Convert it to an integer and print the result.

temperature = 98.6\ntemperature_int = int(temperature)\nprint(temperature_int)

This code creates a variable temperature and assigns it the value 98.6. The value is then converted to an integer using the int() function and stored in the variable temperature_int. The print() function then outputs the integer value.
